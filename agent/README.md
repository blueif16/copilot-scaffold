# Omniscience Agent

LangGraph agents for the Omniscience interactive learning platform.

## Setup

```bash
# Create virtual environment with Python 3.12+
python3.12 -m venv .venv
source .venv/bin/activate

# Install dependencies
pip install -e ".[dev]"
```

## Running Tests

```bash
pytest
```

## Development

```bash
# Start LangGraph dev server
langgraph dev
```
